## Getting Started

### Prerequisites

In order to run this projecto you will need:

- Docker

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/JonasAndrade12/CustomerAPI.git
   ```
2. Run docker-compose
   ```sh
   docker-compose up --build -d
   ```
